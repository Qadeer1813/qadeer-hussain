Birth to Death Rate Ratio Prediciton Using Linear Regression


The Birth to Death Ratio prediction was made using all regions, by years from 1950 to 2023. As you can see on the left, this graph is for the African region, plotted from 1950 to 2023. The Regression Line can tell us if the Birth to Death ratio increases or not.

The source of the data was downloaded from the UN Population Website.

This data was cleaned and processed using the following:
  • Python
  • Jupter Notebook
  • Pandas
  • Numpy
  • Matplotlib
  • Scikit Learn(Sklearn)
